# AgentHub Marketplace

**Community Package Marketplace** - Publicação e descoberta de packages.

## Responsabilidades
- Catálogo de packages públicos
- Publicação de packages
- Instalação de packages
- Rating e reviews
- Busca e filtros

## API
- GET /api/v1/marketplace/packages
- POST /api/v1/marketplace/packages/{id}/install
- POST /api/v1/packages/{id}/publish
